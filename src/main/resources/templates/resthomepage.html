REST Homepage

(Test with PostMan/etc.) Here are all URLS for the App. URLS work like: https://questionnaireapp.herokuapp.com/allanswers SO WRITE https://questionnaireapp.herokuapp.com + END POINT (EXAMPLE/allanswers)

All Questions

Method: GET Endpoint: /api/questions

JSON Example: 

		
		[{
			"questionText": "Miten kurssien työelämäyhteys tuotu esille?",
			"questionType": "text"
		},
		{
			"questionText": "Oletko suorittanut kursseja",
			"questionType": "text"
		},
		{
			"questionText": "Mikä näistä kuvaa parhaiten Haaga-Helian kurssitarjontaa",
			"questionType": "text"
		}]

Question By Id

Method: GET Endpoint: /api/questions/{id}

JSON Example: 

		
		{
			"questionId" : 5,
			"questionText" : "Ovatko tilat viihtyisät?",
			"questionnaire" : {
				"questionnaireId" : 1
		}
		}

Get options

Method: GET Endpoint: /api/options

JSON Example: 

		
		[{
			"question" : {
				"questionId" : 5
		},
			"optionId" : 7,
			"optionText" : "valinta1"
		},
		{
			"question" : {
				"questionId" : 5
		},
		{
			"optionId" : 8,
			"optionText" : "valinta2"
		}]

Get option by Id

Method: GET Endpoint: /api/options/{id}

JSON Example: 

		
		{
			"question" : {
			"questionId" : 5
		},
			"optionId" : 7,
			"optionText" : "valinta1"
		}

Get options by Question Id

Method: GET Endpoint: /api/questions/{id}/options

JSON Example: 

		
		[{
			"question" : {
				"questionId" : 5
		},
			"optionId" : 7,
			"optionText" : "valinta1"
		},
		{
			"question" : {
				"questionId" : 5
		},
			"optionId" : 8,
			"optionText" : "valinta2"
		}]

URL for save answer

Method: POST Endpoint: /saveanswer

JSON Example: 

		
	[ {
  "answerId" : null,
  "answerText" : "Campusonline.fi:n kautta",
  "question" : {
    "questionId" : 7
    }
  },
  {
  "answerId" : null,
  "answerText" : "Virtuaalisesti",
  "question" : {
    "questionId" : 8
  }
} 
]

URL for fetching all Answers from database

Method: GET Endpoint: /allanswers

JSON Example: 

		
		
		
		[
    {
        "answerId": 33,
        "answerText": "Kurssilla on käyty yritysvierailulla",
        "questionId": 7
    },
    {
        "answerId": 34,
        "answerText": "Campusonline.fi:n kautta",
        "questionId": 8
    }
		]

			"questionnaireId": 2,
			"questionnaireTitle": "Tilojen viihtyisyys2",
			"questionnaireDescription": "Tutkitaan tilojen xx2"
