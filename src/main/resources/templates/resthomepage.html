REST Homepage

(Test with PostMan/etc.) Here are all URLS for the App. URLS work like: https://questionnaireapp.herokuapp.com/allanswers SO WRITE https://questionnaireapp.herokuapp.com + END POINT (EXAMPLE/allanswers)

All Questions

Method: GET Endpoint: /api/questions

JSON Example: 

		
		[{
			"questionText": "Miten kurssien työelämäyhteys tuotu esille?",
			"questionType": "text"
		},
		{
			"questionText": "Oletko suorittanut kursseja",
			"questionType": "text"
		},
		{
			"questionText": "Mikä näistä kuvaa parhaiten Haaga-Helian kurssitarjontaa",
			"questionType": "text"
		}]

URL for save answer

Method: POST Endpoint: /saveanswer

JSON Example: 

		
		[{
	  "answerId" : null,
	  "answerText" : "tilat ovat viihtyisät",
	  "question" : {
	    "questionId" : 4
		}, 
		{
	  "answerId" : null,
	  "answerText" : "tilat ovat modernit",
	  "question" : {
	    "questionId" : 5
		}
		}]

URL for fetching all Answers from database

Method: GET Endpoint: /allanswers

JSON Example: 

		
		[{
	        "answerId": 6,
	        "answerText": "tilat ovat viihtyisät"
	            },
	    {
	        "answerId": 7,
	        "answerText": "tilat ovat modernit"
	    }]

		[{

			"questionnaireTitle": "Kurssitarjonta",
			"questionnaireDescription": "Tutkimme Haaga-Helian kurssitarjonnan työelämäyhteyttä sekä opiskelijoiden tyytyväisyyttä kurssivalikoimaan"
			},
		{
			"questionnaireId": 2,
			"questionnaireTitle": "Tilojen viihtyisyys2",
			"questionnaireDescription": "Tutkitaan tilojen xx2"
		}]
