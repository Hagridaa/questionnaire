(Test with PostMan/etc.) Here are all URLS for the App. URLS work like: https://questionnaireapp.herokuapp.com/allanswers SO WRITE https://questionnaireapp.herokuapp.com + END POINT (EXAMPLE/allanswers)

All Questions

Method: GET Endpoint: /api/questions

JSON Example: 

		
		[{
			"questionId": 6,
			"questionText": "Miten kurssien työelämäyhteys on tuotu esille?",
			"questiontypeName": "checkbox"
		},
		{
			"questionId": 7,
			"questionText": "Oletko suorittanut kursseja",
			"questiontypeName": "checkbox"
		},
		{
			"questionId": 10,
			"questionText": "5. Kuinka todennäköisesti suosittelisit Haaga-Heliaa? (1 = en suosittele lainkaan, 10 = suosittelen lämpimästi)",
			"questiontypeName": "Skaala 1-10"
		}]

Question By Id

Method: GET Endpoint: /api/questions/{id}

JSON Example: 

		
		{
			"questionId" : 10,
			"questionText" : "5. Kuinka todennäköisesti suosittelisit Haaga-Heliaa? (1 = en suosittele lainkaan, 10 = suosittelen lämpimästi)",
			"questionnaire" : {
				"questionnaireId" : 1
		}
		}
